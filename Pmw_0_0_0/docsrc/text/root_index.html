Pmw
Python megawidgets

Pmw is a toolkit for building high-level compound widgets in Python using the Tkinter module.

It consists of a set of base classes and a library of flexible and extensible megawidgets built on this foundation. These megawidgets include notebooks, comboboxes, selection widgets, paned widgets, scrolled widgets and dialog windows.

Download Pmw.0.8.5 as tar.gz file (with full documentation)

If you are not sure what to do with a "tar.gz" file, or if you are having trouble unpacking the Pmw distribution, read Getting started with Pmw (also included in the download).

Pmw also contains an interface to the powerful Blt graph widget. An excellent tutorial and reference, "A User's Guide to Pmw.Blt" written by Bjørn Ove Thue and Hans Petter Langtangen, is available. You can also download the full HTML document for local viewing.
